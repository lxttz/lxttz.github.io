Hexo 主题推荐(25个主题总有一个适合你的！！！！)

Hexo 博客系统的主题有很多选择，你可以在 这里 找到它们。

本文主要介绍几个比较好用的主题。

（1）NexT

NexT 主题的特点是简洁，提供三种外观风格，但同时也功能齐全，而且插件丰富，因此是 Hexo 主题中非常流行的一个。
GitHub：https://github.com/next-theme/hexo-theme-next
项目官网：https://theme-next.iissnan.com
推荐值：⭐⭐⭐⭐⭐

（2）Fluid

Fluid 一款 Material Design 风格的 Hexo 主题，由 Fluid-dev 负责开发与维护。

GitHub：https://github.com/fluid-dev/hexo-theme-fluid
项目官网：https://hexo.fluid-dev.com
推荐值：⭐⭐⭐⭐⭐

（3）Butterfly

Butterfly 是 Jerry 在 hexo-theme-melody 主题的基础上进行开发，支持多种美化魔改、多功能组件及静态部署集成的 Hexo 主题。

GitHub：https://github.com/jerryc127/hexo-theme-butterfly
项目官网：https://butterfly.js.org
推荐值：⭐⭐⭐⭐⭐

（4）Anzhiyu

Anzhiyu
简洁明了的 Hexo 主题，设计师风格，一款基于hexo-theme-butterfly修改的主題。
在线演示：https://blog.lixiaomu.fun/
主题仓库：https://github.com/anzhiyu-c/hexo-theme-anzhiyu
官方：https://docs.anheyu.com/
使用文档（主题简介 | 安知鱼主题官方文档）：https://docs.anheyu.com/intro.html
推荐指数：⭐⭐⭐⭐⭐

(5) Diaspora

一款基于 WordPress 移植的 Hexo 主题，适合喜欢摄影，影评，乐评和玩弄文字的你，干净，清新； 响应式，Ajax，更多好玩的等你来发现。
Github 地址：https://github.com/Fechin/hexo-theme-diaspora
Github stars: 1.3K (as of July 2024)
在线演示：http://fech.in/
安装教程：https://github.com/Fechin/hexo-theme-diaspora?tab=readme-ov-file

（6）Acorn

Acorn 是一个免费且开源的 Hexo 主题，和其他主题不一样，Acorn 是适用于企业站点的主题，可适用于中小型企业门户或工作室等展示类网站的简单建设。

GitHub：https://github.com/zhwangart/hexo-theme-acorn
项目官网：https://acorn.imaging.xin
演示：https://acorn.imaging.xin/
使用文档：https://acorn.imaging.xin/docs/index.html

结余

关于哪个主题最好
选择主题之前，一定要明确自己搭建博客的需求。

如果你不能锁定你的需求，挑再好看的主题都白搭。所以，一定要知道自己的目标和需求。

比如搭建博客的目标是为了保存平时写的笔记，那找一个类似于「博客园」那样界面清晰的、目录结构简洁明了的主题最适合你。毕竟，输出技术文档才是你真正的目的。

搭建博客的目标，也不一定只是写技术文档，有的人的目标是做个网站玩一玩，平时也不写东西，即使是心血来潮写点东西，最终也没有人看（自己更不看）；还有的人是想在自己的网站上放点照片啊，心得啊，游记啊之类的东西……

无论如何，一定要找到适合自己的，那才是最好的。
